# Projets-NSI
Projets réalisés en Première et Terminale NSI

### Première

* **PRENSI_Projet_2_Générateur_QCM :** Nombres binaires, C2, IEEE754, Python, Utilisation d'une interface graphique Tkinter

* **PRENSI_Projet_5_Site_Club_Informatique :** Développement web (HTML/CSS/JavaScript) et architecture client-serveur (PHP)
* **PRENSI_Projet_6_Prévoir_Survivants_Titanic :** Utilisation de l'algorithme KNN et traitement de données CSV
* **PRENSI_Projet_7_Game_Hub :** Projet Libre : Développement web avec Flask
